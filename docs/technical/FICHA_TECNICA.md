# 📄 Ficha Técnica Oficial - AgroGo v0.2.5 Stable

Este documento constituye la especificación técnica formal del ERP Agrícola de **ChopCode Solutions**.

---

## 🏗️ 1. Identificación del Producto
*   **Nombre Comercial:** AgroGo.
*   **Segmento:** Agrotech / ERP Rural.
*   **Fabricante:** ChopCode Solutions.
*   **Lead Dev:** Brandon Daza Cerquera.

---

## 💻 2. Stack de Desarrollo
*   **Lenguaje:** Dart 3.x.
*   **Framework:** Flutter 3.12+ (Canal Stable).
*   **Base de Datos Local:** Isar Database (NoSQL de alto rendimiento).
*   **Gestión de Estado:** Riverpod con generación de código.
*   **Inyección de Dependencias:** Riverpod Providers.

---

## 🛰️ 3. Infraestructura y Sensores
*   **Posicionamiento:** GPS/GNSS con filtrado de precisión milimétrica.
*   **Mapas:** Google Maps SDK con soporte para capas híbridas y offline.
*   **Persistencia:** Sistema de archivos local cifrado (path_provider).
*   **Voz:** Speech-to-Text integrado para dictado de registros.

---

## 🛠️ 4. Capacidades Destacadas
1.  **Offline-First:** Funcionamiento al 100% sin conexión a red celular.
2.  **Motor GIS:** Cálculo automático de áreas mediante algoritmos de trigonometría esférica.
3.  **Cross-Module Automation:** Vínculo automático entre recolección, beneficio y bodega.
4.  **Bento Dashboard:** Interfaz de métricas consolidadas en tiempo real.

---

## 🔑 5. Gestión de APIs y Servicios
*   **Google Maps Platform:** Renderizado satelital.
*   **Geolocator API:** Acceso a hardware de ubicación.
*   **Isar Inspector:** Herramienta de depuración de datos en tiempo real.
