# 🎨 Manual de Identidad Visual - AgroGo

Este documento define las directrices estéticas y la paleta de colores para garantizar una experiencia visual profesional en la plataforma **AgroGo**.

---

## 1. Concepto de Marca
AgroGo proyecta una imagen de **Productividad, Naturaleza y Tecnología**. El color predominante es el verde esmeralda, que rinde homenaje a los cafetales colombianos y a la esperanza del campo tecnificado.

## 🚥 3. Estilo de Interfaz (UI)
*   **Bordes**: Uso de un radio de curvatura de **28px a 32px** para tarjetas y contenedores (Estilo amigable).
*   **Glassmorphism**: Efecto de desenfoque y transparencia en los encabezados para crear una sensación de profundidad premium.
*   **Sombras**: Sombras sutiles (`blurRadius: 20, opacity: 0.05`) para elevar los elementos interactivos.

---

## ✒️ 4. Tipografía
*   **Títulos**: Peso `900` (Black) para jerarquía contundente.
*   **Cuerpo**: Peso `Medium` para facilitar la lectura bajo condiciones de sol intenso.
*   **Formateadores**: Uso de fuentes monoespaciadas para valores numéricos y financieros COP.

---

## 🌓 5. Adaptabilidad
La interfaz está optimizada para **Modo Claro (Arena)**:
*   Fondo: `#F9FBF9` (Gris verdoso ultra claro para reducir la fatiga visual).
*   Superficies: Blanco puro `#FFFFFF` para tarjetas de datos.

---

## 📊 6. Estructura de Dashboard y Mapeo (Minimalismo Pro)
La aplicación utiliza una filosofía de diseño **"Thumb-First"** (alcance del pulgar) y composición tipo **Bento Box**:

*   **Dashboard Bento**: Métricas agrupadas en dominios lógicos (Patrimonio, Producción, Recursos) con jerarquía visual clara.
*   **Mapeo Minimalista**:
    *   **Indicadores Flotantes**: Área calculada ubicada en la zona superior para no interferir con el dibujo.
    *   **Controles Inferiores**: Selectores de modo y acciones de guardado agrupados abajo para facilitar la operación con una sola mano en campo.
    *   **Feedback Dinámico**: Uso de micro-interacciones (checks verdes, bordes de colores) para confirmar estados de validación.
